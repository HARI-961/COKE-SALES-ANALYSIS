# Coke Sales Analysis Project

## Objective
To analyze sales performance across months, regions, brands, and retailers using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Data Visualization

## Key Insights
- July recorded highest sales.
- West region contributes highest revenue.
- Coca-Cola is top performing brand.
- Sodapop is highest contributing retailer.

## Strategic Actions
- Plan inventory based on seasonality.
- Focus on high-performing regions.
- Strengthen retailer partnerships.
